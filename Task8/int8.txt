1Ô∏è‚É£ Difference between read(), readline(), readlines()

‚úî Reads complete file
‚ùå Not suitable for very large files

‚úî Reads one line at a time
‚úî Best for large files


readlines()

‚úî Reads all lines into a list
‚úî Useful for line-based processing
